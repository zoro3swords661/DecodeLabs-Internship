The analyzer inspects the sender, links, attachments, and language,
then returns a Close / Warn / Block verdict.
How the verdict maps to action — every event ends in a definitive outcome
SAFE→ Close
Sender verified, no triggers, normal attachment. Close — no action needed.
SUSPICIOUS→ Warn User
Some indicators, unconfirmed. Warn the user and verify out-of-band before acting.
MALICIOUS→ Block & Escalate
Clear indicators. Block the sender domain, report, and escalate to security.
About the data & method. This tool applies detection rules derived from documented phishing patterns (sender/display-name mismatch, right-to-left URL root analysis, URL-shortener detection, dangerous attachment types, and psychological keyword triggers). It analyzes whatever message you paste in — it does not contain a hidden database of real emails. The loadable samples are illustrative examples reconstructed from the attack types shown in the Project 3 brief, provided so the analyzer can be demonstrated live.
